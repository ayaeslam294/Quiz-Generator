 📘 Study Buddy – Quiz Generator

This project helps you generate multiple-choice quizzes (MCQs from study materials.
Upload a TXT, PDF, or DOCX file and get questions automatically.

---

 🚀 Features

 Supports TXT / PDF / DOCX file formats.
 Two modes:

 API Version (using OpenAI API for smart question generation).
 Offline Version (basic questions without API).
 Choose quiz size: 10 / 20 / 30 questions.

---

🖥️ How to Run

You can run this project in different environments:

VS Code → Run the script directly with Python.
JupyterLab / Notebook → Run the cells step by step.
Google Colab** → Upload the notebook and use `files.upload()` to test with your own documents.

---

⚙️ Installation

Install dependencies:

```bash
pip install -r requirements.txt
```
